# Linux

```
Linux is an operating system.
An operating system is software that allows us to use a computer.

Just like:

Windows

macOS

Linux is also an operating system, but it is open-source, which means anyone can see, use, and modify its code.

```

## What does Linux do?

```
Linux controls how the computer works.
It helps the computer to:

Run applications

Manage memory and CPU

Store and read files

Communicate with hardware like keyboard, disk, and network

Without an operating system like Linux, a computer cannot be used.

```

## Linux has two main parts

```
Linux is made of two main parts:

1) Kernel

The kernel is the core of Linux.

It talks directly to the hardware

It controls CPU, memory, and devices

Applications cannot access hardware directly

They must go through the kernel

You can think of the kernel as the manager of the system.

2) User Space

User space is what we actually use.

It includes:

Command line (like terminal)

Applications

System tools

Services

User space talks to the kernel, and the kernel talks to hardware.

```

## What is a Linux Distribution?

```

Linux usually comes as a distribution, also called a distro.

A Linux distribution is:

Linux kernel

Plus tools, applications, and package manager

Examples of Linux distributions:

Ubuntu

Debian

Red Hat

Different distributions use the same Linux kernel but may look and work slightly differently.

```

## Why is Linux popular?
```

Linux is very popular because:

It is free and open-source

It is stable and secure

It runs very well on servers

It is widely used in cloud, DevOps, and containers

Most cloud servers and supercomputers run on Linux.

Linux in real life

Linux is used in:

Servers

Cloud platforms

Docker and Kubernetes

Android phones

Networking devices

Even if we don’t see Linux directly, we use it every day.

Simple summary

To summarize:

Linux is an operating system

The kernel is the core that controls hardware

User space is where users and applications run

Linux is open-source, secure, and powerful

It is widely used in servers and cloud computing

```
